## 📝 Notes

- This repository contains only built/compiled files
- Source code is maintained in a private repository
- Deployments are automated via GitHub Actions
- Environment variables are configured in GitHub Secrets

## 🚀 Deployment

Deployments are triggered automatically when:
- Changes are pushed to the main branch
- Manual workflow dispatch

## 🔧 Configuration

Environment variables are configured in GitHub repository settings:
- Go to Settings → Secrets and variables → Actions
- Add all required environment variables
